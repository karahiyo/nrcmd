# Nrcmd

A command line tool for NewRelic.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'nrcmd'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install nrcmd

## Usage

```
$ nrcmd list-apps --config /path/to/config_file
```
